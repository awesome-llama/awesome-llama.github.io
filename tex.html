You are probably here for the textures, click the button below to copy them to your clipboard.

Then, go back to the project and paste them in to the input box that should've popped up. It will lag the game so hit enter as soon as you paste it, you don't need to wait to see the pasted text appear. Please note that there is a chance that this does not work fully. If your clipboard has a text length limitation, the entire texture might not've been copied. I don't have a solution for this.


How does it work?

The textures the game uses are images of varying sizes. I use a custom image format that consists entirely of printable ASCII characters. 94 of them, to be exact. The 95th character is whitespace, which I also use to separate the textures from each other! This image format can support 24-bit RGB data and compress it decently. I generate this format with a Python script and the project itself has its own script to read it.
