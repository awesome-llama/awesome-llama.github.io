Quiz maker!

Click the green "Run" button at the top!

You can generate questions and then retrieve the save code! Paste it back in to play!

Instructions:

- Click the green "run" button at the top of the screen.
- Now, load up a quiz save! - Try typing in savecodes abc, uav, or taf.
- Paste in the save code. The save code should be sentences/words, each separated by a unicode "middle dot" (Â·). This allows the program to read the save and separate it into a list of items that the quiz runs off.
- Alternatively, you can put in an "x" to indicate you want to create your own.

If creating a quiz:
- Follow the prompts and enter in the details.

If playing a quiz:
- Answer the questions. Some quizzes require precise answers (being case-sensitive and detects all symbols). Others, however do not, and you can put in whatever you want, as long as the text is there and spaces are too, the answer would be correct.
- When answering a question, the quiz will tell you if it correct ot not. If not, the answer will be given.
- At the end, you may see a score out of questions answered. (Not all quizzes do this)
- If you want to stop the quiz at any time, just input nothing and press enter.
